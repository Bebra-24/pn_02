## 📋 О проекте

**АгроКонтроль** - это информационная система для автоматизации процессов управления производством средств защиты растений. Система объединяет в единую цифровую среду технологов, лаборантов и аппаратчиков, обеспечивая прослеживаемость каждой партии продукции от сырья до готового продукта.

### 🎯 Основные возможности

| Модуль | Функции |
|--------|---------|
| **Модуль технолога** | Управление продукцией, рецептурами, технологическими картами, заказами, партиями, настройка экструдера |
| **Модуль лаборатории** | Контроль качества сырья и готовой продукции, ввод результатов анализов, формирование протоколов |
| **API сервер** | REST API с JWT-аутентификацией, 25 эндпоинтов |

## 🚀 Быстрый старт

### Требования

| Компонент | Версия |
|-----------|--------|
| Windows | 10/11 |
| .NET SDK | 8.0 |
| SQL Server | 2019 Express или выше |
| Visual Studio | 2022 |

### Установка

```bash
# 1. Клонирование репозитория
git clone https://github.com/yourusername/AgroControl.git

# 2. Настройка базы данных
cd AgroControl/e
dotnet ef database update

# 3. Запуск API сервера
dotnet run --project e/e.csproj

# 4. Запуск клиентских приложений
# Модуль технолога
dotnet run --project AgroControlClient/AgroControlClient.csproj

# Модуль лаборатории
dotnet run --project AgroControlLaboratory/AgroControlLaboratory.csproj
